# 🚀 LeetCode Company-Wise Interview Questions

A curated collection of **company-wise LeetCode questions** to help you crack coding interviews at top tech companies.

This repository is designed for **DSA preparation**, **placement practice**, and **technical interview readiness**.

---

## 📌 About

This repository contains structured lists of problems from LeetCode categorized by companies.

It helps you focus on **real interview questions** frequently asked in companies like:

* Amazon
* Google
* Microsoft
* Meta
* Adobe
* Flipkart
* And many more...

---

## 🧠 Why This Repo?

* 🎯 Focus on **company-specific preparation**
* 📊 Practice **frequently asked questions**
* ⚡ Save time (no need to search manually)
* 📈 Improve **problem-solving & pattern recognition**
* 🧩 Useful for **OA + Interviews**

1. Choose your **target company**

3. Start solving problems one by one
